# Project 01: Playlist Analyzer

* Author: Mike Taylor
* Class: CS121 Section 4-9
* Semester: Fall 2020

## Overview

Creates a playlist of three song objects and does computations.

## Compiling and Using

To compile, execute the following command in the main project directory:
```
$ javac Playlist.java
```

Run the compiled class with the command:
```
$ java Playlist
```

You will be prompted for song titles, artists, albums, and play time in mm:ss, 
and it will tell you the song closest to 4 minutes, the average play time, and
display the songs in a formatted way from shortest to longest.
